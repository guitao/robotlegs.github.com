Overview

Robotlegs is an ActionScript Application Framework. It offers: Dependency, Module, Command and View management.

Robotlegs 2

Robotlegs 2 is currently in beta and is going to replace v1 very soon.

See Robotlegs v1 in action

There are several live examples for you to take a look at. These are all part of the Robotlegs Demo Bundle which is available for download/forking on Github. Please note: These example might be quite out of date.
